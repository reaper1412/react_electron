# Basic Template for using Electron with React

## Requirements:
* [node.js](https://nodejs.org/en/download/)

### install all dependencies:

    npm install

## Build React app:

    npm run build

## How to run:

    npm run electron

## Portable package:

    npm run dist

## Developed using VS Code:
Download VSC [here](https://code.visualstudio.com/#alt-downloads)
